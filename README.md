# node版本要求
由于本项目中使用到fs的部分函数需要兼容node v14.14.0版本，因此使用此脚手架时需要确保您的node版本在v14.14.0及以上。
# 使用说明：
#### 全局安装
~~~javascript
npm i youth-cli -g
~~~
#### 创建项目
~~~javascript
youth-cli create <projectName>
~~~
